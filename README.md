# BookClub

 # Overview
  The Book Club App is a dynamic web application designed to simulate a book club environment. It allows users to explore books, view detailed information, and engage through interactive features. The application demonstrates core front-end development concepts including API integration, state handling, and DOM manipulation.

 # Key Features
  API Integration – Fetches book data dynamically from a public API
  Book Listing – Displays books with title, author, and description
  Detailed View – View complete information of selected books
  Dark Mode – Toggle theme with persistence using Local Storage
  Shuffle Functionality – Randomly reorder books using JavaScript logic
  Chat System – Send and display messages dynamically

 # Tech Stack
  Frontend: HTML, CSS, JavaScript
  Concepts Used:
  DOM Manipulation
  Asynchronous JavaScript (Fetch API, Async/Await)
  Event Handling
  Local Storage
  Template Literals
  Type Conversion
  Variable Scoping

 # API Details
  Books API: https://www.freetestapi.com/api/v1/books
  CORS Proxy: https://api.allorigins.win/raw?url=[API]

 # How It Works
  Fetches book data from the API asynchronously
  Renders a list of books dynamically on the UI
  Allows users to select a book for detailed view
  Provides theme switching with saved preferences
  Enables book shuffling using randomized sorting logic
  Supports basic user interaction through a chat interface
